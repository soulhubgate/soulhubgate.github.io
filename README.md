# SoulHub - Personality & Psychology Platform

SoulHub is a comprehensive, bilingual (English/Arabic) web application designed to help users explore their personalities, emotional patterns, and psychological traits through various interactive assessments and an extensive knowledge library.

## 🌟 Features

### 🧠 Interactive Assessments
- **Clinical / Personality Styles**: Based on DSM-5 concepts (e.g., Narcissism, Borderline, etc.).
- **MBTI (16 Personalities)**: Jungian archetype assessment measuring E/I, S/N, T/F, J/P dimensions.
- **Love Languages**: Discover how you prefer to give and receive love.
- **Big Five Traits**: Scientific assessment of Openness, Conscientiousness, Extraversion, Agreeableness, and Neuroticism.
- **Dark Triad**: (Coming Soon/In Progress)
- **EQ & Attachment Styles**: (Coming Soon/In Progress)

### 📚 Psychology Library
- In-depth articles explaining various personality types and psychological concepts.
- Bilingual content support (English & Arabic).
- Content expanded to provide deep insights (500-1000 words per topic).

### 🎨 Modern UI/UX
- **Glassmorphism Design**: Sleek, modern interface using Tailwind CSS.
- **Responsive**: Fully optimized for mobile and desktop.
- **Bilingual Support**: Instant toggle between English (LTR) and Arabic (RTL) with full content translation.
- **Dynamic Visualizations**: Radar charts and progress bars to visualize personality profiles.

## 🛠️ Tech Stack
- **Frontend**: HTML5, Vanilla JavaScript
- **Styling**: Tailwind CSS (CDN), Custom CSS animations
- **Charts**: Chart.js for data visualization
- **Icons**: FontAwesome
- **Fonts**: Google Fonts (Inter for English, Cairo for Arabic)

## 📂 Project Structure
- `index.html`: Landing page.
- `tests.html`: Directory of available tests.
- `blog.html`: Psychology library/article viewer.
- `clinical.html`, `mbti.html`, `love.html`, etc.: Individual test applications.
- `js/`:
    - `main.js`: Core logic for navigation, language toggling, and shared utilities.
    - `articles.js`: Content database for the library.
    - `translations.js`: UI string translations.

## 🚀 How to Run
1.  Clone the repository or download the files.
2.  Open `index.html` in any modern web browser.
3.  No build step or server required (Client-side only).
